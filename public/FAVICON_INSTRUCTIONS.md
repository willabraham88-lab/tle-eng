# Favicon Setup Instructions

## Files Created
✅ `/public/favicon.svg` - SVG version created

## Files Needed (Manual Export)

You need to export your TLE logo image in the following formats and place them in the `/public` folder:

### Required Favicon Files:
1. **apple-touch-icon.png** - 180x180px - For iOS home screen
2. **favicon-32x32.png** - 32x32px - Standard favicon
3. **favicon-16x16.png** - 16x16px - Small favicon
4. **logo.png** - 512x512px - For structured data/SEO
5. **og-image.jpg** - 1200x630px - For social media sharing (should include logo + tagline)

### Quick Export Guide:

#### Option 1: Use Online Tool (Easiest)
1. Go to https://realfavicongenerator.net/
2. Upload your TLE logo image
3. Download the generated favicon package
4. Copy the files to your `/public` folder

#### Option 2: Manual Export
Using your image editor (Photoshop, Figma, etc.):
- Export as PNG with transparent background
- Create each size listed above
- Ensure the logo is centered and has some padding

#### Option 3: Use the provided logo
The logo image you provided can be:
1. Saved as different sizes using an image editor
2. Uploaded to a favicon generator
3. Exported directly from your design tool

### Social Media Image (og-image.jpg)
Create a 1200x630px image with:
- Your TLE logo
- Company name: "TELEMETRY & LOGIC ENGINEERING"
- Tagline: "ENGINEERING MEETS EXCELLENCE"
- Background matching your brand colors (teal/blue gradient)

This image will appear when someone shares your website on Facebook, LinkedIn, Twitter, etc.
